# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-03-05

### Added
- Initial release
- LobHub Icons component for Angular
- Support for standalone components
- Support for module-based integration
- GitHub Actions workflows for automated icon updates
- Sample demo application
- Complete TypeScript support
- Documentation in English and Chinese

### Features
- Easy icon integration with `<lobehub-icon>` component
- Support for custom styling and sizing
- Tree-shakeable icon imports
- Automatic icon generation from @lobehub/icons-static-svg
